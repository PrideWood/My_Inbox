# 环境部署

首先，需要注册 github 账号，并下载 github desktop 软件，这是我们后续用来进行调试、同步的基础。之后，访问下面链接，向下滑有详细的中文版使用说明，其实到这里，你就关闭本推文，可以自行搭建了。

> github.com/RayeRen/acad-homepage.github.io

# 正式搭建

下面我们给出详细的操作流程

## 拷贝数据库

点击界面右上角 Fork 选项，将数据库完整的复制到你的仓库中，此处需要对其进行重命名，请**务必**以下面格式命名，这也是将来你访问主页的直接链接：

> **你的 github 昵称**.github.io 比如，你的 githun 昵称为 123456，需要将这个资料库命名为 123456.github.io

![Image](https://mmbiz.qpic.cn/mmbiz_png/LbooKwBKE4SR13FFJJUG7tnpMJWhdagyd1iaCmTJiasfCRrEpPDKdcJicolBIB3E2wTEgsQqKia6eicfQs7V3ic6kyRQ/640?wx_fmt=png&from=appmsg&randomid=nkilyr54&watermark=1&tp=webp&wxfrom=5&wx_lazy=1)

这样，数据库就搭建好了，接下来只需要对里面的内容进行修改即可。

## 内容修改

主页的内容修改分为两个部分，分别是左边的头像、学校、邮箱等基本信息和右边的详细介绍；在网页中直接点击文件即可进行修改，并记得随时保存。

### 基本信息

基本信息的介绍在**config.yml**文件中修改，如下图所示，作者在这里给出了相当多的可编辑项，你甚至可以在谷歌学术的主页设置为 CNKI 的学术主页：

![Image](https://mmbiz.qpic.cn/mmbiz_png/LbooKwBKE4SR13FFJJUG7tnpMJWhdagyuXV02ic1BUvic3t85x3tZ08uicj6hGLoAa90OR5IjnMYG8IdT1N2QsrzA/640?wx_fmt=png&from=appmsg&randomid=4iyzx5ie&watermark=1&tp=webp&wxfrom=5&wx_lazy=1)

### 详细介绍

详细介绍的修改在**pages**文件夹的**about.me**中，整个详细介绍分为了七个部分，只需要逐一对其进行修改就可：

> About Me; News; Publications; Honors and Awards; Educations; Invited Talks; Internships

![Image](https://mmbiz.qpic.cn/mmbiz_png/LbooKwBKE4SR13FFJJUG7tnpMJWhdagyauygjob6LklYmKBajfNnaSPYbfI8jyBiav67r5rKwviam2h8n32Pd98g/640?wx_fmt=png&from=appmsg&randomid=c9t7yzn8&watermark=1&tp=webp&wxfrom=5&wx_lazy=1)

### 图片设置

图片主要涉及头像和成果展示，头像部分，作者提供了下面网站进行生成，个人成果的展示则需要大家自行裁切：

> https://redketchup.io/favicon-generator

需要强调的是，所有设置好的图片应保存到**images**文件夹下，并且调用的时候需要注意其名称，比如，我们将个人头像命名为*android-chrome-512x512.png*，那么**config.yml**中的头像设置就如下所示：

> avatar: “images/android-chrome-512x512.png”

### 内容同步

修改过程中或修改完毕后，记得及时在 github desktop 中点击*Fetch orign*进行同步，这样，就可以在“**你的 github 昵称**.github.io”的网址中查看到自己的个人主页了。
