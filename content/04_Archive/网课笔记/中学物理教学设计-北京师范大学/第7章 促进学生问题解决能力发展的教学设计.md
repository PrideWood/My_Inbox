## 7.1 中学物理问题解决教学

> 发展独立思考和独立判断的一般能力，应当始终放在首位，而不应当把获得专业知识放在首位。——爱因斯坦

### 一、物理问题分类

物理教学中：
- 按编制形式分
- 按问题所涉及对象或适用规律分
- 实践性物理问题、探究性物理问题

| 分类标准 | 问题类型                |
| ---- | ------------------- |
| 开放程度 | 封闭性物理问题 vs.开放性物理问题   |
| 题目结构 | 结构良好物理问题 vs.结构不良物理问题 |
| 抽象程度 | 抽象物理问题 vs.原始物理问题     |
| 认知方式 | 常规性物理问题 vs.非常规性物理问题  |

### 二、物理问题解决教学

传统习题教学与物理问题解决教学对比

|      | 传统习题教学           | 物理问题解决教学                        |
| ---- | ---------------- | ------------------------------- |
| 问题类型 | 封闭性物理问题          | 两种类型的物理问题                       |
| 教学目标 | 重“知识”，轻“能力”      | 三维目标                            |
| 教学方法 | 单一（传授示范式）        | 灵活多样（合作式、探究式）                   |
| 教学环境 | 教师主导；学生感觉枯燥，失去兴趣 | 以学生为主导，教师适时地引导、启发；课堂氛围活跃、学生积极参与 |

## 7.2 中学生解决物理问题的常见困难

1. 认知结构不完善
2. 解题方法的匮乏
3. 思维障碍

## 7.3 中学物理问题解决教学的内容和策略

### 一、创设合适的物理问题

要求：
1. 问题的设置应围绕教学目标，重、难点而展开；
2. 问题的设置应考虑学生现有的知识水平及能力，难度适当；
3. 问题的设置应有利于激发学生兴趣、积极思考、主动探索。

### 二、选择合适的教学策略

> 方法是解题的步骤，还是物理原理、本质呢？

1. 示范引导式教学策略
2. 小组合作式教学策略：组内合作、组间交流、师生互动
3. 探究式教学策略
