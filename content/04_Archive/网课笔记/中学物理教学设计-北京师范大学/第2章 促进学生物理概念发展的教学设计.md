## 什么是大概念？

大概念（big ideas)：能将众多的科学知识联为一致整体的科学学习的核心，包括**共通概念**（crosscutting concepts）和**学科核心概念**
- 共通概念：
	- 模式
	- 因果关系：机制与解释
	- 尺度、比例和数量
	- 系统和系统模型
	- 能量与物质
	- 结构与功能
	- 变化、连续性和稳定
- 学科核心概念，联系 [[2022年版科学课程标准解读 - 喻伯军]]
	- 物质科学的四个核心概念，2013
		- 物质的结构和特性
		- 相互作用、稳定性和变化
		- 能量及及其转变
		- 波作为能量和信息携带者

## 2.1 中学物理概念教学的内容分析与学情分析

### 2.1.1 内容分析——物理概念

- 物理概念的内容分析包括： 
	- 物理概念的意义（为什么引入这个概念）
	- 物理概念的内涵 
	- 物理概念的外延 
	- 物理概念的关联 
	- 物理概念的发展

| 举例  | 加速度                                                        |
| --- | ---------------------------------------------------------- |
| 意义  | 加速度是描述速度变化快慢的物理量，用于精细刻画质点的机械运动。                            |
| 内涵  | 加速度等于质点速度的变化率，是矢量，方向与速度变化量方向一致。                            |
| 外延  | 加速度适用对象仅限于物体的平动。                                           |
| 关联  | 加速度是连接运动与相互作用的桥梁。                                          |
| 发展  | 加速度概念在高中力学中匀变速直线运动、牛顿定律、匀速圆周运动和简谐振动四个部分不断发展，每一部分实现的学习目标不同。 |

### 2.1.2 学情分析

#### 前概念

- 学生在正式学习物理知识之前，头脑中已存在相关认识和思维方式。
- 了解学生前概念的方法
	- 阅读相关研究成果；
	- 咨询有经验的教师；
	- 通过前测和访谈等方式了解学生的前概念。
- 前概念的典型特征
	- 泛性：存在于各学科及其分支中。
	- 移性：对新的知识结构的建立会产生重要影响。

【举例】加速度学情分析
1. 学生对加速度存在误解，如认为加速度是“增加的速度”。
2. 学生已学习过速度概念，对矢量概念有了解。
3. 学生在建构速度概念时采用了比值定义法和极限思想。

## 2.2 中学物理概念教学目标和重难点的确定

【举例】加速度教学目标和重难点

```
# 加速度教学目标

## 知识与技能

1. 知道加速度是描述物体速度变化快慢的物理量；
2. 理解加速度概念，会区别速度、速度变化和速度变化率；
3. 了解加速度的矢量性，会根据速度与加速度方向的关系判断运动性质；
4. 会利用加速度定义式计算匀变速直线运动的加速度；
5. 会利用 v-t 图象计算加速度的大小。

## 过程与方法

1. 经历用比值定义法建立加速度概念的过程；
2. 经历用极限思想，在平均加速度的基础上，建立瞬时加速度的过程。

## 情感态度与价值观

通过加速度与实际生产、生活的联系，体会物理学的实用性，培养学生的学习兴趣。

# 加速度教学重难点

加速度是力学中的重要概念，也是高一学生较难懂的概念。以下是本节教学的重点和难点：

- 区分“速度大”、“速度变化大”、“速度变化得快”，理解速度、速度的变化量、速度的变化率的物理意义，建立加速度的概念；
- 由于学生对矢量只有初步认识，因此，加速度方向与速度方向的关系，以及在直线运动中用正、负号表示加速度方向的理解也是教学中的难点。
```

## 2.3 中学物理概念教学的过程设计和策略应用

物理概念的教学过程大致包含三个环节：
1. 创设物理情境，引入物理概念
2. 运用科学思维方法，引导学生建立物理概念
3. 选择具体问题，运用物理概念

举例：“加速度”的引入
- 录像：飞机、汽车、摩托车比赛（认知冲突）
- 类比学过的速度是如何定义的，鼓励学生自己去定义加速度（比值定义法）

## 2.4 针对前认知的两类教学策略
