### Avoid 10 Common Mistakes

1. Add specific endings when the noun is feminine or plural.
2. Avoid literal translation.
3. Switch the places of subject and object in some cases.
4. Use the right articles.
5. Contract prepositions in some cases. (*du = de + le ; des = de + les ; au = à + le ; aux = à + les ; de la, à la don't contract*)
6. Remove the article in negative sentences.
7. Clarify passé composé (*one-time event in the past*) & imparfait (*what a person used to do over a period of time in the past*).
8. Use "c'est" (*used before a masculine adjective describing a general condition / used before un/une/le/la*) and "il/elle est" (*used when there is no article*) in the right circumstances.
9. Put the adjectives in the right positions. (*adjectives usually come after the nouns, exceptions: bon, mauvais, petit, grand, joli*)
10. Pay attention to "les faux amis" (false cognates).

### Example Sentences

**Français**

1. Il est très amusant. / Elle est très amusante. / Elles sont très amusantes.
2. Je suis intéressée par la mode. / La mode m'intéresse.
3. Tu me manques. // Je les lave.
4. La chaise est petite. // Le mur est vert. // Nous faisons un brunch tous les dimanches.
5. Je vais au marché. // Je vais à la plage.
6. Il n'y a plus de confiture dans le frigo. // Il n'y a pas de pain sur la table.
7. J'ai fait mes devoirs hier soir. // Je faisais mes devoirs tous les jours.
8. C'est beau ici. // C'est une bonne danseuse. // Elle est grande.
9. C'est une jolie fille. // C'est une robe rouge.
10. Je suis actuellement en vacances. // Je vais à la bibliothèque. // J'ai acheté ce roman dans une librairie.

**English**

1. He is very funny. / She is very funny. / They are very funny.
2. I'm interested in fashion. / Fashion interests me.
3. I miss you. // I wash them.
4. The chair is small. // The wall is green. // We have brunch every Sunday.
5. I'm going to the market. // I'm going to the beach.
6. There is no more jam in the fridge. // There is no bread on the table.
7. I did my homework last night. // I was doing my homework every day.
8. It's beautiful here. // She's a good dancer. // She's tall.
9. She's a pretty girl. // That's a red dress.
10. I'm currently on vacation. // I'm going to the library. // I bought this novel at a bookshop.
