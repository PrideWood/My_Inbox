## **说**出一般现在时的变位

- 末尾只有三种不同的发音：
	- je, tu, il/elle, ils/elles + 去掉末尾 -er 的音
	- nous + 去掉末尾 -er 的音 + 鼻音 on
	- vous + 和原形一样的音

## **写**出一般现在时的变位

|           | marcher  |
| --------- | -------- |
| je        | marche   |
| tu        | marches  |
| il/elle   | marche   |
| ils/elles | marchent |
| nous      | marchons |
| vous      | marchez  |

## 省音与连读

- j'aime
- tu aime
- il/elle aime
- ils/elles aiment
- nous aimons
- vous aimez

## 哑音 H
