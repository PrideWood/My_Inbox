## 法语数字难点

### 法语数字进位制

- 一百 cent

| 千级         | 百万级              | 十亿级               |
| ---------- | ---------------- | ----------------- |
| mille      | **un** million   | **un** milliard   |
| dix mille  | dix million**s** | dix milliard**s** |
| cent mille | cent millions    |                   |

### 法语数字的 s

- quatre-vingt**s**
- quatre-vingt-un
- deux/trois/... cent**s**
- deux cent un
- deux million**s**
- deux million**s** dix
- deux milliard**s**
- deux milliard**s** dix

### 法语数字的两个 1

- un / une
- vingt et un garçons / vingt et une filles

### 法语数字连读

- un ami / une amie

### 法语小数

- 用逗号作为小数点
- 小数部分位数较少时读法和整数部分相同，不单独读出小数部分的每个数字；小数部分位数较多时可单独读出每个数字
	- 1,5  un **virgule** cinq
	- 3,14 trois virgule quatorze
	- 3,14159 trois virgule un quatre un cinq neuf
- 由于逗号被用作小数点，在表示大数字时，每三位不用逗号隔开，而用空格，如 1 000 000 000 un milliard

### 法语分数

有对应单词的分数：
- 1/2 : un demi
- 1/3 : un tiers ; 2/3 : deux tiers
- 1/4 : un quart ; 3/4 : trois quarts

其他分数：
- 分子基数词，分母序数词
	- 1/5 : un cinquième
	- 3/8 : trois huitième
- 分母较大时，读出分数线，分母也用基数词
	- 1/97 : un **sur** quatre-vingt-dix-sept
	- 50/253 : cinquante **sur** deux cent cinquante-trois

### 法语百分数

- % : pour cent
	- 2% : deux pour cent
	- 100% : cent pour cent
- ‰ : pour mille
	- 3‰ : trois pour mille
