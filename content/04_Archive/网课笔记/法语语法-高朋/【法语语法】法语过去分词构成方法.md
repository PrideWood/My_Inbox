## 以 -er 结尾变为 -é

- habiter : habité
- passer : passé
- parler : parlé
- aller : allé

## 以 -ir 结尾变为 -i

- finir : fini
- choisir : choisi
- partir : parti
- sortir : sorti

## 变过去分词后以 -u 结尾

- venir : venu
- voir : vu
- vouloir : voulu
- pouvoir : pu
- recevoir : reçu
- savoir : su
- devoir : dû
- boire : bu
- croire : cru
- lire : lu
- connaître : connu
- vendre : vendu
- descendre : descendu
- entendre : entendu

## 变过去分词后以 -is 结尾

- prendre : pris
- apprendre : appris
- comprendre : compris
- mettre :mis

## 变过去分词后以 -it 结尾

- faire : fait
- dire : dit
- écrire : écrit
- traduire : traduit

## Particuliers

- être : été
- avoir : eu  \[y]
- naître : né
- mourir : mort
- offrir : offert
