## 为什么要学习管理思想

管理思想及其发展
- 管理思想是人们对管理过程中发生的各种关系的认识总和，是由一系列观念或观点构成的知识体系。
- 管理思想是指导管理人员从事各项管理活动的路标和蓝图，管理思想的正确与否，直接关系到各项管理活动的效率与效益。
- 随着内外部环境的变化和人们对于管理认识的深入，管理思想也随之发生着相应的演变，并由此形成了若干类不同的管理思想。

## 经验管理思想

- 基本观点：管理的有效性取决于管理者个人的素质。要学会管理必须基于管理实践。
- 代表人物：彼得·德鲁克 (Peter Drucker）和明兹伯格 (Henry Mintzberg)
	- 德鲁克《卓有成效的管理者》《二十一世纪的管理挑战》
	- 明兹伯格《经理工作的性质》《管理者而非 MBA》

## 科学管理思想

- 基本观点：管理的有效性不仅仅取决于管理者个人的经验，更重要地是要依据科学的方法来管理活动和组织。
- 代表人物：
	- “科学管理之父”弗雷德里克·泰勒——科学管理理论；《科学管理原理》
		- 管理要走向科学
		- 劳资双方的心理革命
	- “现代管理之父”亨利·法约尔——一般管理理论；《工业管理与一般管理》
	- “组织理论之父”马克斯·韦伯——官僚组织理论。

泰勒四项科学管理原则

| 传统的做法                  | 科学管理的原则                                  |
| ---------------------- | ---------------------------------------- |
| 工人们按各自的经验来开展工作         | 提出工人操作的每一动作的科学方法，以代替过去单凭经验从事的方法          |
| 由工人选择各自的工作，并尽所能地进行自我培训 | 科学地挑选工人，并进行培训和教育，使之成长成才                  |
|                        | 与工人密切合作，以确保所有工作都按照所制定的科学原则行事             |
| 几乎所有的工作和大部分责任都由工人们承担   | 管理者和工人的工作和职责几乎是均分的。管理者应该承担起那些自身比工人更胜任的工作 |

官僚组织理论的六大特点：
- 劳动分工
- 权力体系
- 正规选择
- 规章制度
- 非人格化
- 职业导向

## 行为管理思想

- 行为管理思想认为：人不仅是经济人，更是社会人，其劳动生产率受到社会的、心理的和群体的因素的影响。
- 代表人物：
	- 梅约——社会人观点，《工业文明中人的问题》《工业文明的社会问题》
	- 马斯洛——需求层次理论
	- 人性假设理论创始人 麦克雷戈—— X 理论和 Y 理论
- 行为管理思想的**特点**
	- 改变了人们对管理的思考方法；
	- 它把人看作是宝贵的资源，强调从人的作用、需求、动机、相互关系和社会环境方面研宄其对管理活动及其结果的影响，研究如何处理好人与人之间的关系、做好人的工作、协调人的目标、激励人的主动性和积极性，以提高工作效率。

## 定量管理思想

- 有效管理的关键在于度的把握，只有致力于定量化才能真正提高管理的效率与效益
- 三个分支
	- 管理科学
	- 作业管理
	- 信息技术在管理中的运用
- 定量管理思想的核心是把数学、统计学和计算机用于管理决策和提高组织效率。
- 代表人物：赫伯特·亚历山大·西蒙，《管理决策新思维》《管理行为》《决策是如何产生的》
- 定量管理思想的特点
	- 力求减少决策中的个人艺术成分，依靠建立一套决策程序和数学模型来寻求最优答案：各种可行方案均是以效益作为评判的依据：广泛使用计算机作为辅助管理手段。
	- 时代的发展，要求管理人员改进他们的决策方法和管理方法，寻找合理分配和应用资源的更好的方法，因此定量管理思想在管理决策中得到了广泛运用，特别是在计划和控制决策中。

## 权变管理思想

- 权变管理思想认为：世界上不存在普遍适用的最佳的管理理论与方法，每一种管理理论与方法都有其一定的适用范围。
- 权变管理思想主要包括系统理论、权变理论、过程理论等。
- 代表人物及著作：
	- 切斯特·巴纳德的《经理的职能》
	- 权变管理理论创始人 费德勒
	- 卡斯特的《组织与管理：系统方法和权变方法》
	- 孔茨和奥·唐奈合著的《管理学》。
	- 权变管理思想的特点：
		- 在继承以前的各种管理思想的基础上，把管理研究的重点转移到了对管理有重大影响的环境因素的研究，希望通过对环境因素的研究，找到各种管理原则和理论的具体适用场合。
		- 以前的管理思想 if X, then Y → 权变管理思想 if X, then Y, only Z
		- 权变管理思想是与 70 年代以来经济活动的国际化、组织大规模化和环境的复杂多变相适应的。

系统理论重要观点：
- 组织是一个系统
- 组织是一个开放的系统
- 组织的性质不仅仅取决于组织的构成要素，更重要的是取决于组织各种要素之间的关系

## 中国管理思想：理论界

- 我国古代的管理思想“繁星点点”，缺乏系统的整理和提高。
- 我国现代管理思想是在继承战争年代的优良传统，接收旧的大机器工业，和学习苏联经济管理理论的基础上发展起来的，由于是在成功与失败之间、正确与错误之间寻求着自己的发展道路，因而我国现代管理思想的发展带有明显的曲折性和复杂性。
- 中国管理理论界现状
	- 同步跟踪引进西方国家先进的管理理论和方法；
	- 着手系统整理中国古代管理思想；
	- 开始总结中国管理实践中的成功或失败经验；
	- 开始探索中国特色的管理理论和方法的建立。

## 21 世纪环境变化带来的管理挑战

==变幻莫测==
- 谁是你真正的竞争对手？
- 谁是你真正的顾客？
- 为提高竞争力我们还能做什么？

经营定位，商业模式

21 世纪经营管理发展趋势：
- 经营客户化：从市场细分、客户定位开始，以顾客满意为导向，注重服务的多样化和服务品质的提升。
- 组织学习化：注重组织学习能力和创新能力的建设，学习和创新成为 21 世纪经营管理的主旋律。
- 管理信息化：面向知识性员工，注重知识管理和传承，基于互联网和信息技术的管理工具的使用成为必然。

==创新==将成为主旋律

## 运用：管理思想的融合

我们应该：学习他人的先进思想**结合自身的实际情况**进行创造性地运用。

## 本讲小结

- 主要的管理思想流派包括经验管理思想、科学管理思想、行为管理思想、定量管理思想、权变管理思想。
- 要搞好管理，就要深刻领会各种管理思想的精华，借鉴他人的思想，结合自身的实际情况，创造出独特的管理思想与方法。
- 中西方之间由于文化价值观念、发展基础等的不同，在管理的关注主题上也有所不同。
- 在 21 世纪，变化将成为日常行为，相应地，创新将成为 21 世纪管理的主旋律。
