关于课堂教学中的小组合作学习互动状况的观察提纲

## 1.观察时间：

G 小为期一周（11.7-11.14）的观察，每天凡是有小组合作的课程或过程均采取直接观察与非参与性观察。

## 2.观察地点：

在 G 校所有班级中，使用整群抽样抽取 1 个班级进行观察。

## 3.观察对象：

观察这个班进行小组合作学习的指导教师与学生。

## 4.观察员：

研究者所有办公室的所有研究员。

## 5.观察内容：

（根据事情的发展顺序来编写 - 逻辑顺序） 
(1) 教师方面

| 观察点   | 观察问题                                  | 概括词 | 描述  |
| ----- | ------------------------------------- | --- | --- |
| ①选题来源 | 教师的选题是怎么来的？                           |     |     |
| ②教师分组 | 教师是以那种依据来确定学生分组的？                     |     |     |
| ③教师指导 | 教师是怎样进行指导的？<br>各组遇到困难，教师是怎样指点和帮助的？    |     |     |
| ④教师态度 | 教师对学生说话的态度是什么样的？                      |     |     |
| ⑤教师提问 | 教师在小组合作过程中有无重要的提问？                    |     |     |
| ⑥教师评价 | 教师是怎样评价各组学习成果的？<br>教师有没有针对某些个别学生进行评价？ |     |     |
| ⑦教师总结 | 小组合作结束后，教师是怎么做总结的？                    |     |     |
| ⑧其他   |                                       |     |     |

（2）学生方面：

| 观察点    | 观察问题                                                                                                     | 概括词 | 描述  |
| ------ | -------------------------------------------------------------------------------------------------------- | --- | --- |
| ①关于选题  | 学生对选题持什么态度？                                                                                              |     |     |
| ②各组分工  | 各组学生分别是怎样进行内部分工的？<br>是否形成了领导者？<br>领导者与其他组员的关系如何？                                                         |     |     |
| ③调查过程  | 各组是用什么方法进行调查？<br>各组是怎样解决任务的？<br>各组遇到困难是怎么解决的？<br>领导者在这个过程中发挥了什么作 用？<br>各组成员之间是怎么样合作和帮助的？<br>他们的调查时间是否充足？ |     |     |
| ④讨论过程  | 各组讨论时间大概多少？<br>各组是否组织过讨论秩序？<br>各组成员相互会倾听吗？<br>各组遇到不一致的观点是怎么处理的？<br>各组的最终结论是如何形成的？<br>各组学生是否期待教师的加入与指导？   |     |     |
| ⑤形成成果  | 各组成果的最终展示者如何选出？<br>成果展示者的语言是什么样的？<br>组员对他们的成果展示者的态度怎样？<br>各组的成果是什么？<br>各组成员之间怎么看待其他组的最终成果？               |     |     |
| ⑥关于评价  | 各组学生从表情看是否期待教师评价与总结？<br>当教师评价到每个组，每组学生的表情怎样？                                                             |     |     |
| ⑦活动结束后 | 小组合作结束后，学生们有什么变化？<br>学生们对这种学习方式的评价？                                                                      |     |     |

## 6.观察方法：

直接观察  非参与性观察

## 7.观察手段：

（1）录制视频   （2）文字记录

## 8.任务分工：

每个研究员负责观察该班级一个小组，另一位研究员负责观察教师，各自做好相应的记录。
