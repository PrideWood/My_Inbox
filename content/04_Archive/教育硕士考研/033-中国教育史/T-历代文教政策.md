## 近现代

| 时期/领导力量   | 教育改革措施/影响                                                                                                                                                                                                                                                                                              |
| --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 太平天国      | 1. 批判儒学<br>2. 改革文字、文风、科举<br>3. 建立普及教育组织（教堂兼学校、育才书院、育才馆和义学）、融政治思想、道德教育于宗教宣传中<br>4. 开放女子教育                                                                                                                                                                                                               |
| 改革派       | 1. “不拘一格降人才”<br>2. 复兴“经世致用”学风<br>3. “师夷长技以制夷”                                                                                                                                                                                                                                                          |

| 洋务运动      | 三类洋务学堂：外国语（“方言”）学堂、军事（“武备”）学堂和技术实业学堂<br>留学教育：留美幼童、留欧学生                                                                                                                                                                                                                                                 |
| 早期改良派     | 1. 全面学习西学<br>2. 改革科举制度<br>3. 建立近代学制<br>4. 倡导女子教育                                                                                                                                                                                                                                                       |
| 维新派       | 1. 兴办学堂<br>2. 兴办学会与发行报刊                                                                                                                                                                                                                                                                                |

| 资产阶级革命派   | 1. 批判封建教育<br>2. 改良派 vs. 革命派<br>3. 开展革命教育活动：中国教育会、爱国女校、爱国学社；大通师范学堂（徐锡麟）                                                                                                                                                                                                                                 |
| 南京临时政府    | 《普通教育暂行办法》《普通教育暂行课程标准》，中国资产阶级首次以中央政府名义发布的教育文件                                                                                                                                                                                                                                                          |

| 新文化运动     | 1. 抨击传统教育，倡导资产阶级的新教育<br>2. 促进教育观念转变：个性化、平民化、实用化、科学化<br>3. 教育改革<br>	1. 1919 国民教育宗旨：养成健全人格，发展共和精神<br>	2. 发展普及教育<br>	3. 教学内容改革：白话文、国语，注意实用<br>	4. 调整全国师范教育布局（设六大示范区），改革大学                                                                                                                                 |
