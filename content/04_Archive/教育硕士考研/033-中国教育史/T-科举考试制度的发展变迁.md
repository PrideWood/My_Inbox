
| 时期   | 发展进程                                                                                                                                                                                                                                                                                                                                                                              |
| ---- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 隋    | 隋炀帝大业二年（606年）开始设置进士科，标志着科举制的正式产生                                                                                                                                                                                                                                                                                                                                                  |
| 唐    | 唐高祖武德四年（621年）恢复科举，明确规定了贡士的范围、主要条件和县初试州复试后每年十月入贡的程序<br>- 考生来源：生徒；乡贡<br>- 考试科目<br>	- **常科**：<br>		- 常设科目：秀才、明经、俊士、进士、明法、明字、明算<br>		- 非常设科目：一史、三史、开元礼、道举、童子<br>	- **制科**<br>	- 武举<br>- 考试方法：帖经、口试、墨义、策问和诗赋                                                                                                                                                                           |
| 宋    | 殿试始于唐朝武则天，但没有成为制度。开宝六年（973 年），宋太祖在讲武殿亲自主持考试，**殿试**成为常制。三级科举考试制度形成：**州**试（由地方官主持）——**省**试（由尚书省礼部主持）——**殿**试（由皇帝主持）<br>自宋太宗（976－997年在位）以后，科举名额大为扩张。<br>宋太宗太平兴国八年（983 年），开始将殿试成绩**评定等第**，把进士分成三甲<br>“庆历兴学”（1044年）改革科举考试，规定科举考试先策，次论，次诗赋，罢贴经、墨义<br>宋英宗治平三年（1066 年），确定“三年一贡举”<br>“熙宁兴学”（1071年）废除明经诸科，进士科考试罢诗赋、帖经、墨义，试以经义、论、策。<br>“崇宁兴学”（1102年）罢科举，改由学校取士。<br>防作弊：建立新制（锁院制、别头试、糊名法、誊录制） |
| 元    | 每三年举行一次，分为**乡试（行省考试）、会试（礼部考试）和御试（即殿试）三级**，将地方解送考试称之为**乡试**<br>- 民族歧视明显<br>- 规定从《四书》中出题，以《四书章句集注》为答题标准<br>- 科举制度日趋严密                                                                                                                                                                                                                                                               |
| 明    | 明朝科举制度的定式始于洪武十七年（1384年），由礼部向全国各省颁布实行，后来成为==“永制”==。<br>- 确定每逢三年开科考试<br>- 四级考试：童试——乡试——会试——殿试，一直延续到清末科举废除<br>八股文形成于明成化年间（1465～1487年），后成为固定考试文体。<br>“科举必由学校”，学校教育纳入科举体系。                                                                                                                                                                                                             |
| 清    | - 分常科和制科两大类<br>	- 常科：文科、武科、翻译科（满洲翻译、蒙古翻译）<br>	- 制科：博学鸿词科、经济特科、孝廉方正科<br>- 科场舞弊丛生，积重难返<br>- 学校成为科举的附庸：以科举中式为目的、内容空疏无用、管理松弛                                                                                                                                                                                                                                                          |
| 太平天国 | 1853 年定都天京起，太平天国开始利用传统的科举考试形式为革命收罗人才。考试的程序基本沿用明清旧制，但对考试的内容和对象则作了重大改革。                                                                                                                                                                                                                                                                                                             |
| 清末   | 1905 年 8 月，清廷下诏“立停科举以广学校”。乡试、会试一律停考，各省岁科考试也随即停，至此，实行了 1300 年的科举制度终告废止。                                                                                                                                                                                                                                                                                                            |
