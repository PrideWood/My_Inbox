
| 论著           | 主要内容                                 | 核心观点                                                                                                               | 特点和影响                                                  |
| ------------ | ------------------------------------ | ------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------ |
| **《大学》**     | 论述大学教育的纲领，强调大学教育的目的、程序和要求            | - **三纲领**：明明德、亲民、止于至善，表现了层层递进的道德目标  <br>- **八条目**：格物、致知、诚意、正心、修身、齐家、治国、平天下，展示循序渐进的学习过程                             | - 强烈的伦理性和人文色彩  <br>- 对汉以后中国知识分子为学、为人与为政产生深远影响          |
| **《中庸》**     | 探讨性与教、中庸思想、自诚明与自明诚、学习五步骤             | - **性与教**：强调教育保存与发扬善性的重要性  <br>- **中庸**：不偏不倚、和谐为核心  <br>- **自诚明与自明诚**：强调内在天性和外在学习的结合  <br>- **五步骤**：博学、审问、慎思、明辨、笃行 | - 提出了学习过程的基本环节，影响后世学习方法  <br>- 中庸思想体现了儒家和谐观，但也表现出一定保守性 |
| **[[《学记》]]** | 中国最早专门论述教育教学问题的著作，涉及教育作用、制度、原则与方法等内容 | - **教育作用与目的**：教育为政治服务，强调教化的重要性  <br>- **教育制度**：包括学制、学年划分、视学与考试制度  <br>- **教育原则**：预防性、及时施教、循序渐进、长善救失、启发诱导等          | - 被称为“教育学的雏形”  <br>- 提出的原则和方法如“教学相长”对中国教育理论影响深远        |
| **《乐记》**     | 论述乐教的起源、作用及其与礼的关系                    | - **乐的起源**：源于人的情感和思想，是外部世界的反映  <br>- **乐的作用**：通过情感感化，达到教育目的，与礼的强制性形成对比                                             | - 强调音乐教育的感化作用，展现儒家“仁政”“德治”的理想  <br>- 乐教思想成为儒家传统的重要组成部分 |
