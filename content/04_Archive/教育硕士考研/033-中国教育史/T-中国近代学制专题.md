
| 学制     | 时间        | 要点                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| ------ | --------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 壬寅学制   | 1902      | 中国近代第一个以中央政府名义制定的全国性学制系统，具体规定了各级各类学堂的性质、培养目标、入学条件、在学年限、课程设置和相互衔接关系，也**规定了强迫（义务）教育**的内容。未施行。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| 癸卯学制   | 1904      | 三段七级：<br>- 第一阶段为初等教育，分为蒙养院四年，初等小学 5 年，高等小学 4 年，共三级 13 年；<br>- 第二阶段为中等教育，仅设中学堂一级，5 年；<br>- 第三阶段为高等教育，分高等学堂（或大学预科）3 年，分科大学堂 3～4 年，通儒院 5 年，共三级 11～12 年。<br>横的方面除直系各学堂外，另有师范教育及实业教育两个系统。                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|        | 1907      | 开放了“女禁”。**1907 年，学部颁布《女子小学堂章程》和《女子师范学堂章程》，这是我国女子教育在学制上取得合法地位的开始**。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
|        | 1909      | 对中学制度进行调整，实行文、实分科，课程各有侧重。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| 壬子癸丑学制 | 1912~1913 | 民国的第一个学制，较全面地反映了资产阶级对教育的要求。<br>- 学制体系<br>	- 整个教育期限为 17～18 年，纵的方面分为三段四级：初等教育（包括初等小学 4 年、高等小学 3 年两级）；中等教育 4 年；高等教育（也只一级，但分预科 3 年、本科 3～4 年）。此外，下设蒙养院，上有大学院。横的方面有师范教育和实业教育两个系统，师范教育分师范学校和高级师范学校两级，实业教育有甲种实业学校和乙种实业学校两种，分农业、工业、商业、商船等类。<br>	- 总年限缩短了3年<br>	- 男女平等观念<br>	- 取消奖励科举出身，废除保人制度<br>	- 规定一年三学期<br>	- 取消高等学堂，只设大学预科<br>- 课程标准<br>	- 废止读经讲经课<br>	- 提高唱歌、图画、手工、农业等课程地位                                                                                                                                                                                                                                                |
|        | 1915      | 袁世凯为推行复古教育对学制进行修改，再其死后又都予以废除。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| 壬戌学制   | 1922      | 11 月 1 日以大总统令公布了==《学校系统改革案》==，这就是 1922 年的“新学制”，或称==“壬戌学制”==，由于采用的是美国式的六三三分段法，又称“六三三学制”。<br>特点：<br>	（1）根据儿童身心发展规律划分教育阶段。<br>	（2）初等教育阶段趋于合理，更加务实。<br>	（3）中等教育阶段是改制的核心，是新学制中的精粹。<br>	（4）建立了比较完善的职业教育系统。中国近现代教育史上，首次用“职业教育”替代“实业教育”<br>	（5）改革师范教育制度。<br>	（6）在高等教育阶段，缩短高等教育年限，**取消大学预科**，使大学不再担任普通教育的任务，这有利于大学进行专业教育和科学研究。兼顾到了升学和就业的双重需要。<br>此外，还有两条“附则”：<br>	一是注重天才教育，得变通修业年限及课程，使优异之智能尽量发展；<br>	二是注意特种教育。<br>评价：<br>- 一定程度上借鉴了美国的六三三制，但它“并不是盲从美制<br>- 加强了中等教育和职业教育训练，并注意以选科制和学分制来适应教育对象的不同发展水平，还注意发挥地方办教育的积极性，提高师范教育水平，缩短小学教育年限以及初中可单设<br>- 尽管受到进步主义教育思想和美国模式的影响，但有其内在的先进性和合理性<br>- 陆续出台了其他一些配套政策，对中国教育的发展起到了**积极的推动作用**。 |
| 戊辰学制   | 1928      | “戊辰学制”分**组织和系统**两部分，第一部分提出了七项原则：根据本国国情；适应民生需要；增高教育效率；提高学科标准；谋个性之发展；使教育易于普及；留地方伸缩之可能等。第二部分为学校系统，戊辰学制与壬戌学制在结构体系上基本相同，只是在中等教育和师范教育方面有一定的改动。该学制颁布后经过多次局部的增改和调整，越来越重视职业教育。                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
