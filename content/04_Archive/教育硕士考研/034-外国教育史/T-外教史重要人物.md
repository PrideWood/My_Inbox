### 古代

| 人物    | 记忆要点                                                                                                                                                                                                                                                                                                                                                                |
| ----- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 苏格拉底  | - 论教育的意义：使人得到改进<br>- 论教育目的：培养治国人才<br>- 教育的首要任务是培养道德，智慧即德行<br>- 要求掌握广博而实用的知识<br>- 苏格拉底方法：四步骤（讥讽、助产术、归纳、定义）<br>- 健康在于锻炼                                                                                                                                                                                                                                               |
| 柏拉图   | - 政治观：三类人（哲学家、军人、手工业者和农民），四种美德（智慧、勇敢、节制、正义）<br>- 哲学观：追求真实知识、追求共相<br>- 学习即回忆<br>- 《理想国》<br>	- 内容<br>		- 理想国的建立和保持，都依赖于教育，七艺<br>		- 男女受同等教育，做同等职业<br>		- 重视早期教育<br>		- 教育最终目的：促使灵魂转向<br>		- 最早提倡“寓教育于游戏”<br>		- 节制<br>	- 对其评价：<br>		- 积极：对应上述<br>		- 消极：过于强调一致性，拒绝变革<br>- 《法律篇》<br>	- 教育管理：长官、学校、教师<br>	- 课程：取消哲学，最高级的课程：算数、几何、天文<br>	- 首次提到强迫教育<br>	- 比《理想国》更具体的一些内容 |
| 亚里士多德 | - 灵魂三部分<br>- 教育作用：发展人的理性，使天性和习惯受理性的领导，人就能成为有良好德行的人<br>- 白板说<br>- 普遍的公立的教育与教育立法<br>- 论各级教育<br>	- 胎教和婴幼儿：优生优育<br>	- 初等教育：阅读、书写、体育锻炼、音乐和绘画<br>	- 高等教育：吕克昂学园<br>	- 重视兴趣<br>	- 家庭教育：个别教育优于共同教育<br>- 伦理思想：幸福是最高目的、美德就是适度、美德在于实践                                                                                                                                           |
| 西塞罗   | - 《论雄辩家》<br>	- 具备条件：广博的知识、修辞学修养、优雅风度举止<br>	- 培养方法：练习（模拟演说、写作）                                                                                                                                                                                                                                                                                                       |
| 昆体良   | - 《雄辩术原理》<br>	- 论教育与天赋：因材施教、遵循儿童年龄特点<br>	- 德行是雄辩家的首要品质（善良：有识别善恶的能力和遵守法律与坚持正义）<br>	- 学校教育优于家庭教育：激励学生、多方面知识、参与公共活动<br>	- 重视学前教育：能说话前后就开始，双语教育，注意不可使厌恶，让最初的教育成为娱乐<br>	- 教学理论<br>		- 班级授课思想萌芽<br>		- 专业教育建立在广播的知识基础上<br>		- 启发诱导、提问解答<br>		- 劳逸结合<br>		- 要求教师：德才兼备、宽严相济、有耐心、懂教学艺术、因材施教                                                                                  |

### 近现代

| 人物     | 记忆要点                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| ------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 夸美纽斯   | - 论教育的目的和作用<br>    - 培养在身体、智慧、德行和信仰几方面和谐发展的人<br>    - 建立在人具有天赋的 “种子” 之上<br>    - 改变社会道德普遍堕落<br>    - 教育对人的发展<br>    - 重视儿童的差异和后天教育<br>- 论教育适应自然的原则<br>- 论普及教育和统一学制<br>    - 普及教育<br>    - 四个时期<br>- 论学年制和班级授课制<br>    - 学年制<br>    - 班级授课制。按根据学生的年龄和程度分成不同班级<br>- 论教学原则<br>    - 直观性原则、循序渐进原则、启发学生学习愿望与主动性原则、巩固性原则<br>- 论道德教育<br>    - 培养德行看作学校的主要任务之一<br>    - 独立<br>- 教育管理思想<br>    - 督学                                                                                                                                                                                                                                                                                  |
| 卢梭     | - 论人的天性<br>	- 性善论和感觉论<br>- 自然教育理论<br>	- 遵循自然要求，顺应自然本性，反对强制教育干涉儿童自由发展<br>	- 自然的教育、人的教育与事物的教育需方向一致、圆满配合<br>	- 到乡村大自然纯朴环境中对儿童进行教育，采用顺应儿童身心发展特点的教育方法，让儿童主动学习，身心自由发展<br>	- 自然的教育主要针对富人<br>	- 把儿童教育划分成四个阶段：婴儿期（出生后两年）、儿童期（2—12岁）、青年期（12—15岁）、青春期（15—20岁），主张因材施教，认为女子应服从权威，培养节制、顺从品德<br>- 公民教育理论<br>	- 公立教育应由国家单独设立的机构领导与管理<br>	- 公民不论贫富享有平等教育权利，接受同样教育，教育目的是培养爱国者，包括身体训练<br>- 教育思想对后世的影响<br>	- 对儿童心理和教育心理发展有可贵启示<br>	- 影响十八世纪末泛爱主义教育发展<br>	- 成为后来儿童中心主义的思想来源之一                                                                                                                                                                                                                    |
| 裴斯泰洛齐  | - 论教育目的<br>	- 发展人的天性和形成完善的人，使人天赋才能充分和谐发展，成为智慧、有德行、身体强健、能劳动且有劳动技能的人<br>- 论教育心理学化<br>	- 教学内容适合儿童学习心理规律，开设广泛基础知识课程<br>	- 教学程序与学生认识过程相协调，遵循直观性、循序渐进教学原则<br>	- 让儿童成为自己的教育者，适应心理时机，调动自我能动性和积极性，培养独立思考能力<br>- 论要素教育<br>	- 初等学校教育从最简单要素开始，逐渐转至复杂要素，促进人和谐发展<br>	- 要求初等学校通过“要素方法”让学生在德、智、体等方面获基本且均衡发展，详细探讨论述德育、智育、体育及其中“要素方法”<br>- 建立初等学校各科教学法<br>	- 语言教学：从发音教学开始，经单词教学，到语言教学<br>	- 算术教学：先通过实物或教具形成数字概念，按加法、乘法、除法、减法顺序进行整数四则运算<br>	- 论述测量教学、地理教学法等<br>- 教育与生产劳动相结合的思想<br>	- 把劳动教育视为儿童获得独立生活能力、改善贫困生活状况手段，力求在教育实践中实施<br>	- 认为不同社会阶层应受符合各自地位的教育，对劳动教育对象认知存在错误，且劳动教育以劳动为主、学习为辅，难学到实际文化知识                                                                                                 |
| 赫尔巴特   | - 伦理学<br>	- 提出五种道德观念：内心自由、完善、仁慈、正义和公平（报偿），作为维持现存世界秩序的道德标准<br>- 心理学<br>	- 提出“下意识”说法，把全部心理活动归结为观念的游戏，重视统觉理论，强调经验积累和知识传授<br>- 道德教育理论<br>	- 教育目的：区分选择目的（职业相关）和道德目的（培养符合观念的人），道德教育是最高目的<br>	- 教育性教学原则：道德教育借助教学，教学与道德教育同等重要<br>	- 性格训练：德育是核心，通过多种手段进行性格训练<br>- 课程理论<br>	- 经验、兴趣与课程：课程内容与儿童经验和兴趣一致<br>	- 统觉与课程：提出“相关”和“集中”原则，以历史和数学为学科中心<br>	- 儿童发展与课程：按儿童四个发展阶段安排课程<br>- 教学理论<br>	- “教学进程”理论：三种教学方法（单纯的提示教学法、分析教学法和综合教学法）构成“教学进程”<br>	- 阶段教学理论：教学分四个阶段（明了、联想、系统与方法），考虑学生心理状态和兴趣，运用统觉原理，有巩固知识环节，但形式机械，后发展为五段教学法                                                                                                                                                                    |
| 福禄贝尔   | - 论教育的基本原理<br>	- 统一的原则：万物服从同一法则，教育目的是培养人的神性，体现自然、人性和上帝的统一<br>	- 顺应自然的原则：神性是人性本质，人的天性是善的，教育要遵循自然发展道路，重视儿童活动本能及发展<br>	- 发展的原则：人的发展和自然进化过程类似，教育应相符，注重儿童身体和精神两方面发展，重视体育、游戏和劳动等<br>- 幼儿园教育理论<br>	- 幼儿园工作的意义和任务：重视童年生活，真正的教育从幼儿阶段开始，父母应注意心智发展，建立幼儿园帮助家庭合理教育幼儿；幼儿园能发展儿童认识和创造性、练习手的技能、打下观察自然基础，形成良好品质，进行美感教育，还能训练教师、推广经验<br>	- 幼儿园教育方法：重视儿童自我活动与自动性，以及亲身观察、共同游戏等<br>	- 幼儿园课程：建立以活动与游戏为主要特征的课程体系，包括游戏、绘画等多种内容                                                                                                                                                                                                                                                                        |
| 蒙台梭利   | - 论幼儿的发展<br>	- 儿童心理发展与遗传、环境的关系：强调遗传作用，推崇内发论，也重视环境对心理和智力发展的作用<br>	- 儿童心理发展的特点：独特的心理胚胎期、心理具有吸引力、发展具有敏感期、发展具有阶段性<br>	- 儿童观要点及特色：重视早期教育，强调心理发展的节律性、阶段性和规律性，重视生命力冲动，强调环境和教育的合理安排<br>- 论教育的环境及自由、纪律与工作<br>	- 教育的环境：教师、环境和儿童相互作用，基本任务是结合生物目的和社会目的，提供有准备的环境，让儿童潜能自由发展<br>	- 自由、纪律及工作：三者有机联系统一，纪律通过自由而来，工作促进纪律形成<br>- 幼儿教育的内容<br>	- 感官教育：重视感官教育，包括多种感官训练，以触觉为主，有独创教具，遵循循序渐进原则<br>	- 读、写、算的练习<br>	- 实际生活的练习：包括日常生活技能、园艺、手工、体操、节奏动作等练习                                                                                                                                                                                                                                                 |
| 杜威     | - 教育本质论<br>	- 教育即生长：儿童心理发展是本能机能生长过程，教育目的是促进本能生长，批判传统教育压抑儿童个性，强调儿童主动精神<br>	- 教育即生活：教育要与儿童眼前生活融合，让儿童适应生活环境，提出“学校即社会”原则<br>	- 教育即经验的持续不断的改造：经验是核心概念，提出“从做中学”原则，冲击传统书本教学，但忽视系统知识传授有错误<br>- 教育的目的<br>	- 教育过程自身发展就是目的，反对普遍性终极目的，强调教育者和受教育者心中具体目的，但抛开社会影响讲教育目的有片面性，又认为要培养青少年维持民主社会等本领，存在矛盾<br>- 课程与教材<br>	- 批判传统课程：反对以既有知识为中心的课程教材，提出“教材心理化”，活动作业应在课程中占中心地位<br>	- 从做中学：“从做中学”贯穿教学各方面，为学生准备环境，让学生在“做”中习得经验、掌握知识和发展思维<br>- 思维与教学方法<br>	- 提倡反省思维，思维包括五个步骤：感觉到问题暗示、明确问题、提出假设、推断结果确定假设、行动中检验假设<br>	- 教学的五个步骤：提供真实经验情境和连续活动、产生真实问题、占有知识资料观察问题、展开解决方法、检验想法有效性<br>- 教育思想总结<br>	- 围绕“儿童中心”，重视学生主动性和创造性，以“做中学”方式开展，形成实用主义教学思想体系，有积极意义，但否定传统学科课程、仅满足活动课程和个人经验不科学，教学过程基于盲目“做”违背教学规律，导致美国教育质量落后 |
| 苏霍姆林斯基 | - 苏霍姆林斯基的道德教育思想<br>    - 道德教育及早开始：从儿童有意识生活一开始便进行，因儿童幼小时心灵易受情感影响。<br>    - 培养道德习惯：自觉性、自尊心和责任感起重要作用，经历正面教育、自我道德评价、道德成熟三个阶段，各阶段有相应培养重点且逐次发展。<br>    - 培养道德情感：情感是核心，德育要培养对美好与丑恶事物的敏感性，同情心等品质是学校教育基本目标。<br>    - 确立道德信念：道德信念是道德发展最高目标，从培养道德习惯入手，将知识转化为行动来稳定形成信念。<br>- 苏霍姆林斯基道德教育思想的启发意义<br>    - 早抓儿童道德教育：个体道德养成循序渐进，要及早进行并因材施教、因时制宜，依儿童年龄特点选合适教育方式。<br>    - 注重知情意行培养：道德教育含知情意行四方面，各方面相互制约、相辅相成，要全面和谐发展以培养品德。<br>    - 提高学生自我教育能力：德育也是学生自我修养过程，要培养学生自我意识、评价、调控能力，发挥其在品德培养中的主体作用。                                                                                                                                                                                                |
