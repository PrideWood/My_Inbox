
| 教学法/计划     | 主要创立者 | 创立时间/地点            | 主要特点                                                                                                               | 存在的问题                                                 |
| ---------- | ----- | ------------------ | ------------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------- |
| **昆西教学法**  | 帕克    | 昆西学校实验期间           | - 强调儿童中心，尊重其内在能力和需求。<br>- 注重学校的社会功能，将学校视为民主政治和理想家庭的雏形。<br>- 将课程与实践活动联系，围绕核心主题安排相关科目。<br>- 培养儿童的自我探索和创造精神。          | -                                                     |
| **有机教育学校** | 约翰逊   | 1907年，美国亚拉巴马州      | - 教育目标为全面发展人的机体，包括感官、体力、智力和社会能力。<br>- 方法遵循学生自然生长，按年龄分组，课程以活动为主。<br>- 重视社会意识培养，强调合作精神和建设性建议能力。<br>- 反对放纵儿童，注重平衡与纪律。 | -                                                     |
| **葛雷制**    | 沃特    | 1917年，美国印第安纳州      | - 根据杜威思想设立课程，包括学术、工艺、活动和体育四方面。<br>- 采用二重编制法，学生分批交替在教室和其他场所活动。<br>- 废除寒暑假，昼夜开放学校，为更多学生提供教育机会。                       | - 对儿童天然兴趣有较好的激发，但难以兼顾学术的深度学习。                         |
| **道尔顿制**   | 帕克赫斯特 | 1920年，美国道尔顿市       | - 废除课堂教学、课程表和年级制，采用“公约”学习。<br>- 设立学科作业室，学生自主学习，教师指导。<br>- 用表格法记录学生进度。<br>- 强调自由与合作，培养学生独立性与社会意识。                   | - 过于强调个别差异，教师压力大，可能导致放任自流。<br>- 教室完全改为实验室不实际。         |
| **文纳特卡计划** | 华虚朋   | 1919-1945年，美国伊利诺伊州 | - 适应儿童个别差异，结合个别学习和小组活动。<br>- 课程分为两部分：共同知识（自学为主）和创造性活动（小组为主）。<br>- 强调个性发展与社会意识的培养。                                  | - 学科深入学习受影响，实施难度大。                                    |
| **设计教学法**  | 克伯屈   | 美国，20世纪初           | - 教学以儿童有目的的活动为核心，分为生产者、消费者、问题和练习四种设计。<br>- 教学四步骤：决定目的、制定计划、实施计划、评判结果。<br>- 强调教学与生活经验的联系，培养学生主动性、合作精神。              | - 主要适用于生产者设计，缺乏明确知识学习的步骤。<br>- 根据儿童经验组织教学，可能削弱系统知识学习。 |
