
| 国家     | 教育法案/报告         | 提出/实施时间     | 主要内容                                                  | 主要解决的问题                      |
| ------ | --------------- | ----------- | ----------------------------------------------------- | ---------------------------- |
| **英国** | **《哈多报告》**      | 1926年       | - 提出中等教育面向全体儿童的思想。<br>- 明确初等教育的终点和分流制度。               | 加强初等教育与中等教育的联系，推进中等教育普及化。    |
|        | **《斯宾斯报告》**     | 1938年       | - 提出三轨制中等教育：文法中学、现代中学、技术中学。<br>- 设想建立多科性中学。           | 改革中等教育的组织和功能，以适应社会多样化需求。     |
|        | **《巴特勒教育法》**    | 1944年       | - 确立三轨制中等教育结构（文法学校、技术中学、现代中学）。<br>- 推动综合中学模式。         | 确立初等教育与中等教育相衔接的制度，推动教育结构现代化。 |
| **法国** | **《阿斯蒂埃法》**     | 1919年       | - 确立职业技术教育为国家管理的事业。                                   | 实现职业技术教育的规范化与体制化。            |
|        | **“统一学校运动”**    | 1919年（运动开启） | - 建立全国统一学制。<br>- 实施统一入学考试，中学初级阶段改为独立公立学校，与初等教育衔接。     | 消除双轨学制弊端，实现教育公平和初中等教育的统一。    |
|        | **《高等教育方向指导法》** | 20世纪中期      | - 确立“自主、自治、民主参与、多科性结构”的高等教育原则。<br>- 推动大学改组，调整课程与办学方式。 | 确保高等教育结构适应现代社会需求，提升教育质量与公平性。 |
