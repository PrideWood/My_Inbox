
| 理论名称           | 代表人物                                           | 时间         | 贡献（观点）                                                       |
| -------------- | ---------------------------------------------- | ---------- | ------------------------------------------------------------ |
| 德国的格式塔心理学      | 苛勒                                             | 20世纪初      | 强调人类的心理功能一般倾向于处理连贯的实体（格式塔），并通过其结构和联系来理解现象。                   |
| 美国的行为主义        | 约翰·B.华生 (John B. Watson)                       | 1913年      | 只有能够直接观察和测量的东西才是科学的，意识、意义和情感等概念被排除在外。                        |
| 苏联的文化历史理论      | 维果斯基 (Lev Vygotsky)、鲁利亚 (Alexander Luria) 等    | 20世纪20年代中期 | 人类的学习能力基本上是在人类的种系发展、实践工具的使用和高级心理功能的发展中塑造而成的。                 |
| 皮亚杰的建构主义       | 让·皮亚杰 (Jean Piaget)                            | 20世纪20年代   | 关于儿童时期人类智力发展的阶段性理论，区分了学习中增长、变化或重构之间的差异。                      |
| 弗洛伊德和杜威的互补视角   | 西格蒙德·弗洛伊德 (Sigmund Freud)、约翰·杜威 (John Dewey)   | 20世纪上半叶    | 弗洛伊德的“精神分析”概念和实践促使人们注意到了情绪的心理影响；杜威的“儿童中心取向”及“做中学”的思想产生了深远影响。 |
| 斯金纳的极端行为主义     | B.F.斯金纳 (B.F. Skinner)                         | -          | 人类的行为是由人类机体和环境之间的相互作用决定的，学习完全依赖于输入。                          |
| 人本主义心理学        | 卡尔·罗杰斯 (Carl Rogers)、亚伯拉罕·马斯洛 (Abraham Maslow) | -          | 罗杰斯引入了“以患者为中心”的治疗方法，马斯洛的需求或动机层次理论强调了动机的重要性。                  |
| 体验学习理论         | 戴维·科尔布 (David Kolb)                            | 1984       | 提出了体验学习理论和模型，认为学习是一个有着四个阶段的过程，包含具体经验、反思性观察、抽象概念化和主动实验。       |
| 社会建构主义         | 肯尼斯·格尔根 (Kenneth Gergen)                       | 1991       | 认为学习只能通过社会背景获得理解，学习过程发生在个体内部，但其性质和内容总是由社会领域的关系决定。            |
| 情境学习理论         | 让·莱夫 (Jean Lave)、艾蒂安·温格 (Etienne Wenger)       | 1991       | 认为学习总是在特定的情况下进行的，影响学习过程及结果。                                  |
| 直觉学习、学校学习和专家学习 | 霍华德·加德纳 (Howard Gardner)                       | 1991       | 提出了三种学习方式：直觉学习、学校学习和专家学习，强调直觉学习的重要性。                         |
| 支架、叙述和学习       | 杰罗姆·布鲁纳 (Jerome Bruner)                        | -          | 提出了“支架”概念，强调学习者以叙述的方式表达自我，并把此作为个人学习和发展的核心。                   |
| 活动理论再概念化       | 恩格斯特罗姆 (Yrjo Engestrom)                        | 1987       | 提出了三种学习类型，包括扩展学习，强调人类活动系统的结构模型。                              |
| 建构发展视角         | 罗伯特·凯根 (Robert Kegan)                          | 1982       | 描述了学习带来的一般人类发展，提出了五个连续阶段的终身学习过程。                             |
| 终身学习视角         | 彼得·贾维斯 (Peter Jarvis)                          | 1980年代后期   | 关注社会条件与学习者个人发展之间的联系，特别是在成年期。                                 |
