
| 学派         | 代表人物                          | 核心观点                                                       |
| ---------- | ----------------------------- | ---------------------------------------------------------- |
| **行为主义**   | 巴甫洛夫（Ivan Pavlov）             | 条件反射理论，经典条件作用：行为通过刺激与反应的联系形成（如狗对铃声分泌唾液的实验）。                |
|            | 斯金纳（B.F. Skinner）             | 操作条件作用：通过强化或惩罚塑造行为（如鸽子按键获得食物的实验）。                          |
|            | 桑代克（Edward Thorndike）         | 联结主义：学习是刺激与反应之间形成联结的过程，提出“效果律”。                            |
|            | 华生（John B. Watson）            | 行为主义的创始人，强调行为可以通过外部环境的控制进行预测和塑造。                           |
| **认知主义**   | 皮亚杰（Jean Piaget）              | 认知发展阶段理论：儿童的认知发展分为四个阶段（感知运动、前运算、具体运算、形式运算），学习是主动构建认知结构的过程。 |
|            | 布鲁纳（Jerome Bruner）            | 发现学习理论：强调通过探索和发现促进学习，提出表象系统理论（动觉表象、影像表象、符号表象）。             |
|            | 奥苏泊尔（David Ausubel）           | 有意义学习理论：学习是通过新的信息与已有知识之间建立联系来实现的，区分机械学习与有意义学习。             |
|            | 维果茨基（Lev Vygotsky）            | 社会文化历史理论：学习在社会互动中完成，最近发展区和内化是关键概念。                         |
| **信息加工理论** | 阿特金森与希夫林（Atkinson & Shiffrin） | 多重记忆模型：信息加工分为感觉记忆、短时记忆和长时记忆三个阶段。                           |
|            | 巴德利（Alan Baddeley）            | 工作记忆模型：强调短期记忆的动态性，提出中央执行系统、语音环路、视觉空间模版等结构。                 |
|            | 加涅（Robert Gagné）              | 信息加工的学习层次理论：提出学习的八个层次和五类学习成果，强调学习的系统设计和教学过程。               |
| **人本主义**   | 马斯洛（Abraham Maslow）           | 需求层次理论：人的行为受基本需求（如生理需求、安全需求等）驱动，最高层次是自我实现。                 |
|            | 罗杰斯（Carl Rogers）              | 自我中心学习理论：学习应以学生为中心，强调个体的自由、自主和潜能发展。                        |
| **建构主义**   | 杜威（John Dewey）                | 强调教育与社会生活的联系，学习是通过实践和反思实现的，提倡以学生为中心的教学。                    |
|            | 维果茨基（Lev Vygotsky）            | （同上）学习通过社会文化背景和互动实现，强调合作学习的重要性。                            |
|            | 皮亚杰（Jean Piaget）              | （同上）认知发展是主动建构知识的过程，提出“同化”与“顺应”概念。                          |
