| 理论      | 基本观点                                                                                        | 评价                                                                                      |
| ------- | ------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------- |
| 强化论     | 人的某种学习行为倾向完全取决于先前的这种学习行为与刺激因强化而建立起来的稳固联系，强化可以增强学习动机。                                        | 强调外部强化对动机的影响，解释了奖励和惩罚如何影响行为的持续性，但忽视了个体的内在心理过程和自主意愿。                                     |
| 需要层次说   | 马斯洛认为人的需要从低到高分为生理需要、安全需要、归属与爱的需要、尊重需要求知与理解的需要、美的需要和自我实现的需要。低层次需要满足后，人们才会追求高层次需要，这些需要影响学习动机。 | 考虑到了人的内在需求对动机的驱动作用，较为全面地阐述了人类动机的复杂性，但在实际应用中，需求层次的划分可能因个体差异和文化背景而有所不同。                   |
| 自我决定理论  | 德西和瑞安强调人是积极的有机体，具有自我决定的倾向，主要包括**自主需要、能力需要和关系需要**。当这些需要得到满足时，个体的学习动机就会增强。                    | 关注个体的内在心理需求对学习动机的重要性，突出自主性在学习中的关键作用，有利于培养学生的内在学习动机，但在实际教育情境中，三种需要的平衡和满足可能较难把握。          |
| 成就动机理论  | 阿特金森认为成就动机分为**力求成功的动机和避免失败的动机**。力求成功者倾向于选择**中等难度**的任务，避免失败者倾向于选择非常容易或非常困难的任务。               | 能够很好地解释个体在面对不同难度任务时的动机差异，但可能过于简化了成就动机的复杂心理过程，并且没有充分考虑社会文化等外部因素的影响。                      |
| 自我效能感理论 | 班杜拉认为自我效能感是指个体对自己是否有能力完成某一行为所进行的推测与判断。自我效能感高的个体更有动力去完成任务。                                   | 强调个体的自我认知对学习动机的关键作用，为培养学生积极的学习动机提供了具体的干预方向，如通过榜样示范等方式提升自我效能感，但自我效能感的测量和判断可能受到多种因素的干扰。   |
| 控制点理论   | 个体对自己生活中发生的事情及其结果的控制源有不同的解释，分为**内控型和外控型**。内控型的人认为自己可以控制结果，外控型的人认为结果是由外部因素决定的，这种观念影响学习动机。    | 有助于理解个体在学习中的责任归因方式对动机的影响，但控制点的划分可能比较绝对，个体在不同情境下可能表现出不同的控制点倾向。                           |
| 归因理论    | 韦纳认为人们对行为成败原因的分析可归纳为六个因素（能力、努力、任务难度、运气、身心状态、外界环境），这些归因方式会影响个体的学习动机。                         | 详细地分析了不同归因方式对动机的影响，为教育者引导学生进行积极归因提供了理论依据，但在实际应用中，学生的归因方式可能受到情绪等多种因素的干扰，并且难以简单地改变。       |
| 成就目标理论  | 主要有两种成就目标，即**掌握目标**（关注知识的掌握和能力的发展）和**表现目标**（关注与他人比较的结果），不同的成就目标影响学习动机。                      | 能够很好地解释学生在学习过程中不同的动机取向，为教育者理解学生的学习动机提供了新的视角，但两种目标并非完全独立，在实际学习情境中可能相互影响。                 |
| 自我价值理论  | 学生的自我价值主要来自于自己的能力，当学生面临可能失败的情境时，会采取各种策略来维护自己的自我价值，这种自我价值保护的需求影响学习动机。                        | 关注学生的心理需求，特别是自我价值感对学习动机的重要性，解释了学生一些消极学习行为背后的心理动机，但在实际教育中，可能会过分强调学生的自我价值保护，而忽视了对失败的正确引导。 |
| 调节聚焦理论  | 个体存在两种调节聚焦方式，即促进聚焦（关注成长、进步和成就）和预防聚焦（关注安全、责任和义务），这两种聚焦方式影响学习动机。                              | 从个体的心理调节角度解释学习动机，为理解个体在学习中的动机差异提供了新的维度，但在实际研究和应用中，两种聚焦方式的界定和测量可能存在一定的困难。                |
