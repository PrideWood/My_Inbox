Kimi 电影推荐：
- [x] 《放牛班的春天》：描述了一位音乐教师如何通过音乐将希望带给一群被社会边缘化的学生。
- [x] 《心灵捕手》：讲述了一个被遗弃数次的孤儿，拥有极高的数学天赋，最终在心理医生的帮助下走出阴影，开启新生活的故事。
- [x] 《三傻大闹宝莱坞》：教育就是要挖掘孩子们本身的潜能，而非填鸭式的“大波轰”。
- [ ] 《卡特教练》：讲述了一位篮球教练如何通过严格的训练方法和关爱，帮助一支贫困学校的篮球队实现突破。
- [x] 《黑板》：在伊朗战火纷飞的库尔德地区，一群教师背着黑板到处找学生授课，以此来养家糊口的故事。
- [x] 《起跑线》：这部印度电影反映了教育问题，演绎现代版“孟母三迁”。
- [x] 《垫底辣妹》：讲述了一个差生的逆袭故事，强调老师只是引路者，关键还是要强大自我。
- [x] 《死亡诗社》：讲述了一位非传统教师基汀如何鼓励他的学生自由思考和追求自己的梦想。
- [ ] 《超脱》：描绘了一位教师如何隐藏自己的痛苦，同时努力鼓励他的学生在一所疯狂的学校中追求知识和生活的意义。
- [x] 《一个都不能少》：讲述了代课老师魏敏芝为了寻找一位辍学学生而踏上了艰难的寻人之旅。
纪录片推荐：
- [x] 《教育能改变吗》：围绕中国教育改革中的热点问题进行完整的梳理，并从历史和文化的高度对中国教育的改革与创新作出思考。
- [x] 《极有可能成功》：介绍了美国加州一所高中正在进行的教育创新尝试，主要采用“项目式学习”的方式进行教学。
- [ ] 《中国学校》：以安徽省一个小镇为缩影，集中拍摄了一组家庭、老师、孩子们一个学年的生活。
- [ ] 《等待超人》：记录了五名学生寻求更好教育却受挫的过程，揭露了美国公立教育存在的种种弊端。
- [x] 《中国门》：讲述了关于高考的“中国式奋斗”的故事，从甘肃会宁、北京、上海三处拍摄，试图以三段不同背景下关于教育的故事，组成一幅当代教育现象集中呈现的现实图景。
- [ ] 《镜子》：中国首部深度探讨家庭情感教育的纪录片。
- [x] 《教育强国》：讲述了中国教育制度的改革和取得的伟大进步。
- [x] 《两百万分钟：一次跨国考察》：中美印，三国教育大比拼，关于孩子教育的话题。
- [ ] 《出路》：讲述了中国教育的现状，以及人们对教育的尊崇和期待。
- [x] 《我们的孩子足够坚强吗》：让 5 名中国老师在英国南部汉普郡的一所中学实施 4 周中国式教学试验的故事。
